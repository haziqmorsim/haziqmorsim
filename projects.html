Internship Project



Project name: Employee Details Website

Language(s): Angular, Typescript


The aim of this project is work on front-end web development using Angular and Bootstrap web framework, specifically Material Design for Bootstrap and Angular (MDBA). I was required to create a website that displays the employees' details by applying the CRUD (Create, Read, Update and Delete) operations to describe user interface conventions that facilitate viewing, searching, and changing information using computer-based forms and reports. The Employee Details website was created using a specific template created by the DWA Team that is used by all Evonik websites. The first page of the website displays the employees list inside an Angular Datatable. The table contains the employee ID, name and position. The search bar is available to filter the input results accordingly and there are “Create New” button, “Edit” button, and “Delete” button to create, edit, and delete an employee respectively.

Final Year Project



Project name: Hospital Appointment Management System

Language(s): MERN Stack


Health Link is a comprehensive hospital appointment management system developed using the MERN (MongoDB, Express.js, React.js, Node.js) stack. This project aims to streamline and enhance the efficiency of the appointment scheduling process within healthcare institutions. The system prioritizes user experience, accessibility, and real-time data management to provide a seamless solution for both hospital and patients. Key functionalities of Health Link include user authentication, appointment creation, modification, and cancellation, as Ill as a comprehensive dashboard for hospital to manage their schedules efficiently. The system employs RESTful APIs for smooth communication betIen the server and the client, ensuring data consistency and reliability. Health Link not only addresses the immediate needs of appointment management but also considers the long-term scalability and adaptability requirements of healthcare institutions. This MERN stack project is designed to enhance the overall healthcare experience by optimizing appointment workflows and fostering better communication between hospital and patients.
